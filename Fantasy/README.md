# О чем эти файлы
Тут я привожу ключевые тезисы космологий различных фантастических миров, зачастую вместе с популярными стереотипами про них (часто возникающими под влиянием фильмов и/или игр).
Ранее хотел, но не имею мотивации или "музы" оформлять это в полноценное эссе с художественным стилем.\
Некоторые заметки написаны другими авторами (текст по миру Мартина -- Кириллом Ферулевым).
